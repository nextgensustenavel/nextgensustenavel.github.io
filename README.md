# NextGen Sustentável

Site institucional do NextGen Sustentável, construído com Astro e publicado no GitHub Pages.

## Desenvolvimento

Requer Node.js 22.12 ou mais recente.

```sh
npm install
npm run dev
```

## Produção

```sh
npm run build
npm run preview
```

O conteúdo de produção é gerado em `dist/`. Novos commits na branch `main` acionam a publicação pelo GitHub Actions.
